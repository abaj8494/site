** notes

it is good to keep notes about version controlling things.

*** git

| prefix | use-case                          |
|--------+-----------------------------------|
| BUG    | bug fix                           |
| DEV    | development tool or utility       |
| DOC    | documentation                     |
| ENH    | enhancement, a new feature        |
| MAINT  | maintainence task                 |
| REL    | release                           |
| STY    | stylistic change                  |
| TST    | addition or modification of tests |
